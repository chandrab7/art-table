# Art of the Table

Event styling & custom gift box business website. A "Midnight Garden" themed static site with blush pink and gold accents on deep navy backgrounds.

## Setup

```bash
git clone https://github.com/chandrab7/art-table.git
cd art-table
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

Alternatively, just open `index.html` directly in any browser.
